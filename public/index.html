D&D Beyond Monster Converter

To get a valid Cobalt token follow these steps:

  1. Login to D&D Beyond
  2. Open the developer console (F12)
  3. Go to the Application tab
  4. In the left sidebar, go to Storage > Cookies
  5. Find the cookie named CobaltSession and copy the value (this is the cobalt token)

This tool is still in early stages of development, if you run into any issues please file a report here please be sure to include the url of hte monster you are trying to convert.
